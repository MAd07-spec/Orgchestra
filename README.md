# Orgchestra
organs and orchestra

## How to run in BlueJ

This project uses JavaFX for the GUI. To run it in BlueJ:

1. Download the JavaFX SDK from https://openjfx.io/ (choose the version matching your Java version).
2. Extract the SDK to a folder, e.g., `C:\javafx-sdk-17.0.2`
3. In BlueJ, go to Tools > Preferences > Libraries
4. Add the JavaFX library path: `C:\javafx-sdk-17.0.2\lib`
5. Alternatively, for the project, right-click the project in BlueJ, select Properties, and add the library path there.
6. Compile and run MainApp.

## Description

An interactive application where you can "play" different organs by clicking buttons or using keyboard shortcuts. Each organ produces a unique sound using synthesized audio or MIDI.

- Heart (H): Plays a heartbeat sound
- Lungs (L): Sustained drone (hold key or button)
- Trachea (T): Similar to lungs
- Diaphragm (D): Plays a sound
- Stomach (S): Sustained sound
- Kidneys (K): Plays a sound
- Intestines (I): Variable frequency sound (drag in button area)
- Appendix (A): Plays a sound
- Vocal Cords (V): Plays a sound

The body image is displayed with invisible buttons over the organs.
